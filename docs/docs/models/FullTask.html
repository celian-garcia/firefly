FullTask

  • name : String
    Display name of the task.
  • description : String
    Description of the task.
  • type : Integer
    The unique ID which identify the type of the task.
  • module : Integer
    The unique ID which identify the module to which the task belongs.
  • id : Integer
    Unique id representing a specific task.
  • state : Integer
    State of the task.
  • user_name : String
    Name of the user who created the task.
  • date : String
    Date and time of the task creation, in iso format.
